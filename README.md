# Probabilites of Allen Interval Relations

Final Year Project — B.A. (Mod.) in Computer Science, Linguistics and a Language
Trinity College Dublin

## Overview

This project extends Allen's Interval Algebra (1983) by introducing probabilistic modeling and simulations to handle uncertainty in temporal relations and their composition. Through an interactive dashboard, users can explore how different birth and death probabilities affect the distribution of Allen relations and analyze the composition of these relations.

## Features

- **Complete Implementation** of Allen's 13 interval relations
- **Compositional Reasoning** with a full composition (transitivity) table
- **Probabilistic Simulations** using birth/death state transitions
- **Statistical Analysis** of relation distributions with hypothesis testing
- **Interactive Dashboard** with three main visualization components:
  - Animated Distribution Evolution
  - Interactive Composition Heatmap
  - Parameter Surface Explorer
- **Real-time Animation** showing how relation distributions converge over trials
- **Rich Visualizations** with 3D surface plots and heatmaps
